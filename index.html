EXAMEN II

COLA:

                   
                
En JavaScript, un método de cola generalmente se refiere a una estructura de datos de tipo FIFO (First In, First Out), donde los elementos se agregan al final y se eliminan
                
algoritmos principio. La implementación más simple usa un array, con métodos como push() para agregar y shift() para eliminar. Es una estructura fundamental y se usa en  
                
muchos y aplicaciones, especialmente cuando es importante mantener el orden de llegada de los elementos.

PILA::

                
El método pila o estructura de datos de pila (también conocida como LIFO, Last In, First Out, o "último en entrar, primero en salir") es un tipo de estructura de datos 
                
donde el último elemento agregado es el primero en ser eliminado.Esta estructura es ampliamente utilizada en programación y tiene muchas aplicaciones prácticas, como 
                
en la ejecución de funciones recursivas, manejo de operaciones matemáticas, y control de flujo de ciertos algoritmos.

LISTA:

                
En JavaScript, una lista se puede representar mediante un array (arreglo). Los arrays son estructuras de datos que permiten almacenar múltiples valores bajo una sola
                
variable. Estos valores pueden ser de cualquier tipo: números, cadenas de texto, objetos, e incluso otros arrays.

DOBLAR:

                
En JavaScript, doblar (o duplicar) los elementos de un array puede referirse a diferentes cosas, dependiendo del contexto:
                
Duplicar cada valor en un array: Si quieres multiplicar cada número de un array por 2, por ejemplo, puedes hacerlo utilizando el método map().
                
Duplicar todo el array: Si quieres duplicar el array en sí mismo (crear una copia del mismo), hay diferentes formas de hacerlo.

CIRCULAR:

                
En JavaScript, cuando hablamos de una estructura circular o array circular, generalmente nos referimos a una lista donde el último elemento apunta
                
de vuelta al primer elemento, creando una "circularidad". Este concepto es útil para simular estructuras de datos como colas circulares o buffers
                
circulares, donde, al alcanzar el final de la lista, el próximo acceso será al principio de la lista.

ARBOL:

                
Un árbol es una estructura de datos que se organiza en forma jerárquica. Está compuesto por nodos conectados entre sí. Cada nodo puede tener un valor
                
y referencias a nodos "hijos". El nodo superior, que no tiene ningún nodo que lo preceda, se llama raíz.
                
Los árboles son útiles para representar relaciones jerárquicas o jerarquías de datos, como los sistemas de archivos de una computadora, las relaciones 
                
familiares o incluso las estructuras de menús.
