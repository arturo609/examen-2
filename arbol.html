Clase Nodo


La claseNodose utiliza para representar cada nodo en el árbol. Cada nodo tiene tres propiedades:
valor: El valor que almacena el nodo.
derecha:Un puntero que apunta al hijo derecho del nodo (es otro nodo o null si no existe).
izquierda:Un puntero que apunta al hijo izquierdo del nodo (también es otro nodo o null si no existe).

Clase Arbol
La clase Arbol representa el árbol binario. Tiene una propiedad ruta que apunta al nodo raíz del árbol (o a null si el árbol está vacío).

Propiedades:
ruta:Es la raíz del árbol. Si el árbol está vacío, ruta es null.
Métodos: isEmpty():Verifica si el árbol está vacío, es decir, si ruta es null.

add(valor):Este método inserta un valor en el árbol binario. Si el árbol está vacío, el primer nodo (ruta) se crea con el valor dado. Si ya existe al menos un nodo, el valor se inserta en el árbol de acuerdo con las reglas del Árbol Binario de Búsqueda (ABB):
Si el valor es menor que el nodo actual, se va a la izquierda.
Si el valor es mayor o igual, se va a la derecha.
Para insertar un valor:
Primero se verifica si el árbol está vacío, y si es así, el valor se coloca como la raíz(ruta).
Si el árbol no está vacío, se comienza desde la raíz y se recorre el árbol comparando el valor a insertar con los nodos existentes. Si el valor es menor que el nodo actual, se mueve a la subárbol izquierdo; si es mayor o igual, se mueve a la subárbol derecho. El proceso continúa recursivamente hasta encontrar una posición vacía (donde izquierda o derecha sea null) para colocar el nuevo nodo.
