class Cola { 
                constructor() {
                    // Se crea un objeto vacío para almacenar los elementos de la cola.
                    this.almacenCola = {}; // Almacenamos los elementos en un objeto.
                    this.inicio = 0;       // Puntero al primer elemento de la cola.
                    this.fin = 0;          // Puntero al siguiente índice donde se agregará el próximo elemento.
                }
            
                // Método para agregar un elemento al final de la cola (enqueue)
                enqueue(elemento) {
                    this.almacenCola[this.fin] = elemento; // Añadimos el nuevo elemento en la posición 'fin' de la cola.
                    this.fin++;  // Incrementamos 'fin' para apuntar al próximo índice donde se agregará el siguiente elemento.
                    return this.almacenCola;  // Retorna el estado actual de la cola.
                }
            
                // Método para quitar y retornar el primer elemento de la cola (dequeue)
                dequeue() {
                    if (this.inicio === this.fin) { // Si el puntero 'inicio' es igual a 'fin', la cola está vacía.
                        return null;  // No hay elementos que devolver, entonces retornamos 'null'.
                    }
                    const elemento = this.almacenCola[this.inicio]; // Guardamos el primer elemento de la cola.
                    delete this.almacenCola[this.inicio];  // Eliminamos el primer elemento de la cola.
                    this.inicio++;  // Incrementamos 'inicio' para apuntar al siguiente elemento.
                    return elemento;  // Retornamos el primer elemento que fue eliminado de la cola.
                }
            
                // Método para obtener el primer elemento de la cola sin eliminarlo (peek)
                peek() {
                    if (this.size() === 0) {  // Si el tamaño de la cola es 0 (está vacía).
                        return null;  // No hay elementos que devolver, retornamos 'null'.
                    }
                    return this.almacenCola[this.inicio];  // Retornamos el primer elemento sin eliminarlo.
                }
            
                // Método para obtener el tamaño de la cola (número de elementos en la cola)
                size() {
                    return this.fin - this.inicio;  // La diferencia entre 'fin' e 'inicio' nos da el tamaño de la cola.
                }
            
                // Método para imprimir todos los elementos de la cola
                print() {
                    return this.almacenCola;  // Retorna el objeto que contiene los elementos de la cola.
                }
            
                // Método para comprobar si la cola está vacía
                isEmpty() {
                    if (this.inicio === this.fin) { // Si 'inicio' es igual a 'fin', la cola está vacía.
                        return true;  // Retorna 'true' si la cola está vacía.
                    }
                    return false;  // Si no, la cola no está vacía, retornamos 'false'.
                }
            }
            
            // Se crea una nueva instancia de la cola llamada 'COLA'.
            const COLA = new Cola();

La clase "COLA" tiene tres propiedades principiales,
this.almacencola: Es un objeto vacio que se utiliza para almacenar lo elementos de la cola.
this.inicio: Es el puntero que maraca el primer elemento de la cola (el que fue agregado primero) este valor comienza en 0.
this.fin: Es el elementp que marca el siguiente indice disponible para agregar un nuevo elemento, este valor tambien comienza en 0.

Los metodos:
*Almacenar el nuevo elemento en this.almacenCola en la pocicion de this.fin.
*Luego Incrementa this.fin para que apunte al siguiente indice disponible para el proximo elemento.
*Retora el objeto completo this.almacenCola para mostrar el estado actual de la cola.

size(); este metodo retorna el numero de elementos en la cola. la calcula como la diferencia entre this.fin y this.inicio marca el siguiente indice disponible y this.inicio marca el primer indice ocupado
isEmpty() Este método devuelve true si la cola está vacía (es decir, si no hay elementos en la cola). Lo comprueba comparando this.inicio con this.fin Si ambos son iguales, significa que no hay elementos en la cola.

enqueue(elemento): Agrega un elemento al final.
dequeue(): Elimina y retorna el primer elemento
peek(): Retorna el primer elemento sin eliminarlo.
size(): Retorna el número de elementos en la cola.
print(): Muestra los elementos de la cola.
isEmpty(): Verifica si la cola está vacía.
